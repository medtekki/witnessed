# Receipts v1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the v1 trust path for verifiable agent action receipts — an agent signs a claim locally, a witness service independently timestamps and countersigns it, the receipt is stored and retrievable, and anyone can verify it offline.

**Architecture:** A TypeScript pnpm monorepo. A shared `core` package holds canonicalization, hashing, crypto, and types. A pure `verifier` package validates receipts and is reused by both clients (offline) and the server (hosted), guaranteeing identical verdicts. A `witness` package holds the witness logic, a `ReceiptStore` interface with an in-memory append-only implementation, and a Hono HTTP service. An `sdk` package handles client keys and issuing/verifying. An `mcp` package exposes `issue_receipt`, `verify_receipt`, `get_receipt` tools.

**Tech Stack:** TypeScript, pnpm workspaces, Node `node:crypto` (Ed25519), `canonicalize` (RFC 8785 JCS), Hono, `@modelcontextprotocol/sdk`, Vitest.

  crv: "Ed25519";
  x: string;
  d?: string; // present only on private keys; never sent to the witness
}

export interface Agent {
  key_id: string;
  public_key: Jwk; // public only (no `d`)
  label?: string;
  on_behalf_of?: string;
}

export interface Action {
  type: string;        // e.g. "email.send"
  description: string; // human-readable
}

/** The signed core. Contains NO id, signatures, or witness (avoids circularity). */
export interface Claim {
  version: 1;
  issued_at: string;          // ISO 8601, agent's claimed time
  agent: Agent;
  action: Action;
  payload_digest: string | null; // base64url sha256 of salient content, or null
  anchor: null;               // reserved for effect-binding (inert in v1)
  prev: null;                 // reserved for multi-agent chaining (inert in v1)
}

export interface Witness {
  witnessed_at: string;   // ISO 8601, authoritative time
  witness_key_id: string;
  witness_sig: string;    // base64url
}

export interface Receipt extends Claim {
  id: string;        // base64url sha256 of canonical(Claim)
  agent_sig: string; // base64url, over canonical(Claim)
  witness: Witness;
}

export interface VerifyResult {
  valid: boolean;
  claim?: Claim;
  reason?: string; // populated when valid === false
}

import canonicalizeFn from "canonicalize";
import { createHash } from "node:crypto";

export function canonical(value: unknown): string {
  const out = canonicalizeFn(value as object);
  if (out === undefined) throw new Error("value is not canonicalizable");
  return out;
}

export const b64u = {
  encode(bytes: Uint8Array): string {
    return Buffer.from(bytes).toString("base64url");
  },
  decode(s: string): Uint8Array {
    return new Uint8Array(Buffer.from(s, "base64url"));
  },
};

export function sha256b64u(data: string | Uint8Array): string {
  const buf = typeof data === "string" ? Buffer.from(data, "utf8") : Buffer.from(data);
  return createHash("sha256").update(buf).digest("base64url");
}
```

import { generateKeyPairSync, createHash } from "node:crypto";
import type { Jwk } from "./types";

export function generateKeyPair(): { publicJwk: Jwk; privateJwk: Jwk } {
  const { publicKey, privateKey } = generateKeyPairSync("ed25519");
  const publicJwk = publicKey.export({ format: "jwk" }) as unknown as Jwk;
  const privateJwk = privateKey.export({ format: "jwk" }) as unknown as Jwk;
  return { publicJwk, privateJwk };
}

/** Returns the public half only (strips `d`). */
export function toPublicJwk(jwk: Jwk): Jwk {
  return { kty: jwk.kty, crv: jwk.crv, x: jwk.x };
}

/** RFC 7638-style thumbprint over the public members, base64url. */
export function thumbprint(jwk: Jwk): string {
  const canonicalMembers = `{"crv":"${jwk.crv}","kty":"${jwk.kty}","x":"${jwk.x}"}`;
  return createHash("sha256").update(canonicalMembers, "utf8").digest("base64url");
}
```

import { sign as nodeSign, verify as nodeVerify, createPrivateKey, createPublicKey } from "node:crypto";
import type { Jwk } from "./types";

export function sign(message: string, privateJwk: Jwk): string {
  const key = createPrivateKey({ key: privateJwk as object, format: "jwk" });
  const sig = nodeSign(null, Buffer.from(message, "utf8"), key);
  return sig.toString("base64url");
}

export function verifySig(message: string, signatureB64u: string, publicJwk: Jwk): boolean {
  try {
    const key = createPublicKey({ key: publicJwk as object, format: "jwk" });
    return nodeVerify(null, Buffer.from(message, "utf8"), key, Buffer.from(signatureB64u, "base64url"));
  } catch {
    return false;
  }
}
```

export function buildClaim(input: {
  issued_at: string;
  agent: Agent;
  action: Action;
  payload_digest: string | null;
}): Claim {
  return {
    version: 1,
    issued_at: input.issued_at,
    agent: input.agent,
    action: input.action,
    payload_digest: input.payload_digest,
    anchor: null,
    prev: null,

export function computeId(claim: Claim): string {
  return sha256b64u(canonical(claim));

export function signClaim(claim: Claim, privateJwk: Jwk): string {
  return sign(canonical(claim), privateJwk);

import { canonical, verifySig, computeId } from "@witnessed/core";
import type { Receipt, Claim, Jwk, VerifyResult } from "@witnessed/core";

/** trustedWitnessKeys maps witness_key_id -> public JWK the verifier trusts. */
export function verifyReceipt(
  receipt: Receipt,
  trustedWitnessKeys: Record<string, Jwk>,
): VerifyResult {
  // Reconstruct the signed claim (strip id, agent_sig, witness).
  const claim: Claim = {
    version: receipt.version,
    issued_at: receipt.issued_at,
    agent: receipt.agent,
    action: receipt.action,
    payload_digest: receipt.payload_digest,
    anchor: receipt.anchor,
    prev: receipt.prev,
  };

  if (computeId(claim) !== receipt.id) {
    return { valid: false, reason: "id does not match claim contents" };
  }
  if (!verifySig(canonical(claim), receipt.agent_sig, receipt.agent.public_key)) {
    return { valid: false, reason: "agent signature is invalid" };
  }

  const witnessKey = trustedWitnessKeys[receipt.witness.witness_key_id];
  if (!witnessKey) {
    return { valid: false, reason: "unknown or untrusted witness key" };
  }
  const witnessMessage = canonical({
    id: receipt.id,
    agent_sig: receipt.agent_sig,
    witnessed_at: receipt.witness.witnessed_at,
  });
  if (!verifySig(witnessMessage, receipt.witness.witness_sig, witnessKey)) {
    return { valid: false, reason: "witness signature is invalid" };
  }

  return { valid: true, claim };
}
```

  get(id: string): Promise<Receipt | null>;

    return this.map.get(id) ?? null;
  }

import { canonical, verifySig } from "@witnessed/core";
import type { Claim, Receipt } from "@witnessed/core";
import type { Signer } from "./signer";
import type { ReceiptStore } from "./store";
import { computeId } from "@witnessed/core";

export interface WitnessDeps {
  signer: Signer;
  store: ReceiptStore;
  now: () => string; // injected clock for testability; returns ISO 8601
}

export interface WitnessRequest {
  claim: Claim;
  agent_sig: string;
}

/** Returns an async witness function bound to its dependencies. */
export function makeWitness(deps: WitnessDeps) {
  return async function witness(req: WitnessRequest): Promise<Receipt> {
    const { claim, agent_sig } = req;

    if (!verifySig(canonical(claim), agent_sig, claim.agent.public_key)) {
      throw new Error("agent signature is invalid");
    }

    const id = computeId(claim);
    const witnessed_at = deps.now();
    const witness_sig = deps.signer.sign(canonical({ id, agent_sig, witnessed_at }));

    const receipt: Receipt = {
      ...claim,
      id,
      agent_sig,
      witness: { witnessed_at, witness_key_id: deps.signer.keyId, witness_sig },
    };

    await deps.store.put(receipt);
    return receipt;
  };
}

import { Hono } from "hono";
import { toPublicJwk } from "@witnessed/core";
import type { Jwk } from "@witnessed/core";
import { verifyReceipt } from "@witnessed/verifier";
import { LocalSigner } from "./signer";
import { InMemoryStore } from "./store";
import { makeWitness } from "./witness";

export interface AppConfig {
  witnessPrivateJwk: Jwk;
  witnessKeyId: string;
  now?: () => string;
}

export function createApp(config: AppConfig) {
  const signer = new LocalSigner(config.witnessPrivateJwk, config.witnessKeyId);
  const store = new InMemoryStore();
  const now = config.now ?? (() => new Date().toISOString());
  const witness = makeWitness({ signer, store, now });
  const trusted: Record<string, Jwk> = { [config.witnessKeyId]: toPublicJwk(config.witnessPrivateJwk) };

  const app = new Hono();

  app.post("/receipts", async (c) => {
    const { claim, agent_sig } = await c.req.json();
    try {
      const receipt = await witness({ claim, agent_sig });
      return c.json(receipt, 201);
    } catch (err) {
      return c.json({ error: (err as Error).message }, 400);
    }
  });

  app.get("/receipts/:id", async (c) => {
    const receipt = await store.get(c.req.param("id"));
    return receipt ? c.json(receipt) : c.json({ error: "not found" }, 404);
  });

  app.post("/verify", async (c) => {
    const receipt = await c.req.json();
    return c.json(verifyReceipt(receipt, trusted));
  });

  return app;
}

import {
  generateKeyPair, toPublicJwk, thumbprint, buildClaim, signClaim,
} from "@witnessed/core";
import type { Action, Jwk, Receipt, VerifyResult } from "@witnessed/core";
import { verifyReceipt } from "@witnessed/verifier";

export interface ClientConfig {
  baseUrl: string;
  privateJwk: Jwk;
  fetchImpl?: typeof fetch;
}

export class ReceiptsClient {
  private readonly publicJwk: Jwk;
  private readonly keyId: string;
  private readonly fetchImpl: typeof fetch;

  constructor(private readonly config: ClientConfig) {
    this.publicJwk = toPublicJwk(config.privateJwk);
    this.keyId = thumbprint(this.publicJwk);
    this.fetchImpl = config.fetchImpl ?? fetch;
  }

  static generateKey() {
    return generateKeyPair();
  }

  async issueReceipt(input: {
    action: Action;
    issued_at?: string;
    payload_digest?: string | null;
  }): Promise<Receipt> {
    const claim = buildClaim({
      issued_at: input.issued_at ?? new Date().toISOString(),
      agent: { key_id: this.keyId, public_key: this.publicJwk },
      action: input.action,
      payload_digest: input.payload_digest ?? null,
    });
    const agent_sig = signClaim(claim, this.config.privateJwk);

    const res = await this.fetchImpl(`${this.config.baseUrl}/receipts`, {
      method: "POST",
      headers: { "content-type": "application/json" },
      body: JSON.stringify({ claim, agent_sig }),
    });
    if (res.status !== 201) {
      throw new Error(`witness rejected receipt: ${res.status} ${await res.text()}`);
    }
    return (await res.json()) as Receipt;
  }

  verify(receipt: Receipt, trustedWitnessKeys: Record<string, Jwk>): VerifyResult {
    return verifyReceipt(receipt, trustedWitnessKeys);
  }
}

import { McpServer } from "@modelcontextprotocol/sdk/server/mcp.js";
import { StdioServerTransport } from "@modelcontextprotocol/sdk/server/stdio.js";
import { z } from "zod";
import type { ReceiptsClient } from "@witnessed/sdk/src/client";
import type { Jwk, Receipt } from "@witnessed/core";

export interface HandlerDeps {
  client: ReceiptsClient;
  trustedWitnessKeys: Record<string, Jwk>;
}

/** Pure, directly-testable tool handlers. */
export function makeHandlers(deps: HandlerDeps) {
  return {
    async issue_receipt(args: { action_type: string; description: string; payload_digest?: string | null }) {
      const receipt = await deps.client.issueReceipt({
        action: { type: args.action_type, description: args.description },
        payload_digest: args.payload_digest ?? null,
      });
      return { receipt };
    },
    async verify_receipt(args: { receipt: Receipt }) {
      return deps.client.verify(args.receipt, deps.trustedWitnessKeys);
    },
  };
}

/** Wires the handlers into an MCP server over stdio. */
export function createMcpServer(deps: HandlerDeps): McpServer {
  const handlers = makeHandlers(deps);
  const server = new McpServer({ name: "receipts", version: "0.0.0" });

  server.tool(
    "issue_receipt",
    { action_type: z.string(), description: z.string(), payload_digest: z.string().nullable().optional() },
    async (args) => ({ content: [{ type: "text", text: JSON.stringify(await handlers.issue_receipt(args)) }] }),
  );

  server.tool(
    "verify_receipt",
    { receipt: z.any() },
    async (args) => ({ content: [{ type: "text", text: JSON.stringify(await handlers.verify_receipt(args as { receipt: Receipt })) }] }),
  );

  return server;
}

export async function start(deps: HandlerDeps): Promise<void> {
  const server = createMcpServer(deps);
  await server.connect(new StdioServerTransport());
}

## Definition of Done (v1)

- An agent can generate a key, issue a receipt via SDK or MCP, and the witness independently timestamps + countersigns it.
- Receipts persist in the append-only store and are retrievable by id over HTTP.
- Anyone holding the trusted witness public key can verify a receipt **offline** with the same `verifyReceipt` the server uses.
- Tampering with any claim field, the agent signature, or the witness signature fails verification.
- `anchor` and `prev` exist in the schema but are inert (`null`), ready for the flight-recorder expansion.
- `pnpm test` is green.

## Explicitly deferred (expansion / roadmap — not in v1)

- Effect-binding (`anchor` validation), multi-agent chaining (`prev`), human-oversight checkpoint receipts.
- Durable storage (SQLite/Postgres), retention/legal-hold, transparency-log checkpoints.
- KMS/HSM-backed `Signer`, key registry service, key rotation/revocation surfaces.
- Public verify web page, Python SDK, REST surface beyond the witness endpoints, auth/rate-limiting/billing.
